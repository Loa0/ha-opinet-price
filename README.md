# 오피넷 주유소 최저가 (HA Opinet Price)

오피넷 API와 Tmap API를 활용하여 주변 주유소의 최저가 정보와 실제 주행거리를 제공하는 Home Assistant 커스텀 컴포넌트입니다.

## 주요 기능

- **주변 주유소 검색**: 설정 위치 기준 반경 내 모든 주유소 검색 (0.5~20km)
- **최저가/주행거리 정렬**: 가격순 또는 Tmap 실제 주행거리순 정렬
- **필터링**: 브랜드(다중선택), 셀프주유소, 고속도로(전체/고속도로만/제외)
- **즐겨찾기**: 자주 가는 주유소를 별도 엔티티로 추가
- **1~10위 개별 센서**: `오피넷 주유소` 기기로 묶어서 관리
- **상세 정보**: 주유소명, 가격, 주소, 브랜드, 거리(직선/주행거리)
- **거리 표시 토글**: 엔티티 상태에 거리 표시 ON/OFF

## 설치 방법 (HACS)

1. HACS → 통합구성요소 → 우측 상단 ⋮ → 커스텀 저장소
2. Repository: `Loa0/ha-opinet-price`, Category: `Integration` 입력 후 추가
3. "오피넷 주유소 최저가" 검색 후 설치
4. Home Assistant 재시작

## 설정 방법

### 초기 등록

| 항목 | 설명 | 필수 |
|---|---|---|
| API 키 | [오피넷](https://www.opinet.co.kr)에서 발급 | ✓ |
| 정렬 순서 | 가격순 / 주행거리순 | ✓ |
| Tmap API 키 | 주행거리순 선택 시 필수 | 조건부 |
| 상품코드 | 휘발유(B027) / 경유(D047) / 고급휘발유 / 등유 / 부탄 | - |
| 위치 엔티티 | `device_tracker` 또는 `person` (미선택 시 HA 기본 위치) | - |

### 옵션 설정 (기기 등록 후)

`설정 → 기기 및 서비스 → 오피넷 주유소 기기 → 설정` 에서 탭으로 진입:

| 메뉴 | 설명 |
|---|---|
| API 키 | 오피넷 API 키, Tmap API 키 변경 |
| 필터 설정 | 브랜드, 셀프, 고속도로, 반경 등 |
| 즐겨찾기 관리 | 주유소명 검색 → 다중 선택 → 별도 센서 생성 |

**필터 설정 항목:**

| 항목 | 타입 | 설명 |
|---|---|---|
| 반경 | 슬라이더 | 0.5~20km |
| 주유소 필터 | 다중선택 드롭다운 | SK에너지, GS칼텍스, S-OIL 등 |
| 셀프주유소만 | 스위치 | ON 시 셀프 주유소만 표시 |
| 고속도로 | 드롭다운 | 전체 / 고속도로만 / 고속도로 제외 |
| 거리 표시 | 스위치 | 센서 상태에 거리 표시 ON/OFF |
| 정렬 순서 | 드롭다운 | 가격순 / 주행거리순 |

**즐겨찾기:** 선택한 주유소가 `sensor.opinet_price_favorite_1`, `_2`, ... 로 생성됩니다.

## Tmap API 키 발급 방법

1. [Tmap API](https://tmapapi.tmapmobility.com/main.html) 접속 → 회원가입/로그인
2. **API Key 관리** → **API Key 발급**
3. 용도: `경로탐색` (Routes) 선택
4. 발급된 `appKey` 값을 Tmap API 키 항목에 입력

> 무료 플랜 기준 월 30,000건 제공 (주유소 30개 × 3시간 간격 ≈ 월 7,200건)

## 센서 정보

`오피넷 주유소` 기기 아래 1~10위 센서가 생성됩니다.

**상태(state) 형식:**
- 거리 ON: `SK에너지: 1,984원 (4.1km)`
- 거리 OFF: `SK에너지: 1,984원`
- 주행거리순 사용 시: Tmap 실주행거리 우선 표시

**속성(Attributes):**
- `주유소명`, `가격`, `주소`, `간략주소`, `브랜드`, `거리`, `순위`

## 업데이트 방법

HACS에서 최신 버전 Redownload 후 **통합구성요소 삭제 → 재등록**을 권장합니다.
