# 오피넷 주유소 최저가 (HA Opinet Price)

> ⚠️ 본 통합구성요소는 한국 Opinet API를 사용하므로 **대한민국에서만** 사용 가능합니다.

오피넷 API와 Tmap API를 활용하여 주변 주유소의 최저가 정보와 실제 주행거리를 제공하는 Home Assistant 커스텀 컴포넌트입니다.

## 주요 기능

- **주변 주유소 검색**: 설정 위치 기준 반경 내 모든 주유소 검색 (0.5~20km)
- **최저가/주행거리 정렬**: 가격순 또는 Tmap 실제 주행거리순 정렬
- **필터링**: 브랜드(다중선택), 셀프주유소, 고속도로(전체/고속도로만/제외)
- **즐겨찾기**: 자주 가는 주유소를 별도 엔티티로 추가
- **지도 카드**: 주유소 위치를 HA 기본 맵 카드에 마커로 표시 (클릭 시 가격·주소·거리 팝업)
- **갱신 최적화**: 갱신시간(1,2,9,12,16,19시 KST) + 이동 시(5~20km)만 호출로 API 절약
- **API 사용량 추적**: 오피넷·Tmap 호출 횟수 실시간 센서, 자동 리셋
- **1~10위 개별 센서**: `오피넷 주유소` 기기로 묶어서 관리
- **상세 정보**: 주유소명, 가격, 주소, 브랜드, 거리(직선/주행거리)
- **거리 표시 토글**: 엔티티 상태에 거리 표시 ON/OFF

## 설치 방법 (HACS)

1. HACS → 통합구성요소 → 우측 상단 ⋮ → 커스텀 저장소
2. Repository: `Loa0/ha-opinet-price`, Category: `Integration` 입력 후 추가
3. "오피넷 주유소 최저가" 검색 후 설치
4. Home Assistant 재시작

> 요구사항: Home Assistant **2024.1.0** 이상

## 다중 인스턴스

휘발유/경유 등 상품별로 분리 모니터링이 필요하면 통합구성요소를 한 번 더 등록하세요.
`설정 → 기기 및 서비스 → 통합구성요소 추가 → 오피넷 주유소` 에서
상품코드만 다르게 설정하여 추가 인스턴스를 생성할 수 있습니다.

> v1.10.15+ 부터 인스턴스 간 unique_id 충돌이 해결되어 안전하게 여러 개 등록 가능합니다.

## 설정 방법

### 초기 등록

| 항목 | 설명 | 필수 |
|---|---|---|
| API 키 | [오피넷](https://www.opinet.co.kr)에서 발급 | ✓ |
| 정렬 순서 | 가격순 / 주행거리순 | ✓ |
| Tmap API 키 | 주행거리순 선택 시 필수 | 조건부 |
| VWorld API 키 | 좌표 정확도 향상 & 일 50회 제한 해제 | - |
| 상품코드 | 휘발유(B027) / 경유(D047) / 고급휘발유 / 등유 / 부탄 | - |
| 위치 엔티티 | `device_tracker` 또는 `person` (미선택 시 HA 기본 위치) | - |

### 옵션 설정 (기기 등록 후)

`설정 → 기기 및 서비스 → 오피넷 주유소 기기 → 설정` 에서 탭으로 진입:

| 메뉴 | 설명 |
|---|---|
| API 키 | 오피넷 API 키, Tmap API 키 변경 |
| 필터 설정 | 브랜드, 셀프, 고속도로, 반경 등 |
| 즐겨찾기 관리 | 주유소명 검색 → 다중 선택 → 별도 센서 생성 |
| 갱신 설정 | 이동 갱신 ON/OFF, 갱신 거리(5~20km), API 사용량 안내 |

**필터 설정 항목:**

| 항목 | 타입 | 설명 |
|---|---|---|
| 반경 | 슬라이더 | 0.5~20km |
| 주유소 필터 | 다중선택 드롭다운 | SK에너지, GS칼텍스, S-OIL 등 |
| 셀프주유소만 | 스위치 | ON 시 셀프 주유소만 표시 |
| 고속도로 | 드롭다운 | 전체 / 고속도로만 / 고속도로 제외 |
| 거리 표시 | 스위치 | 센서 상태에 거리 표시 ON/OFF |
| 정렬 순서 | 드롭다운 | 가격순 / 주행거리순 |

**즐겨찾기:** 검색하여 선택한 주유소가 `sensor.opinet_price_fav_<uni_id>` 로 별도 생성됩니다.
- 반경 밖에 있어도 `detailById.do` API로 실시간 가격 조회
- 즐겨찾기 등록 시 상호명·주소가 영구 저장되어 목록에서 항상 식별 가능
- 즐겨찾기 삭제 시 관련 엔티티(센서, 위치트래커) 자동 정리

**갱신 설정 항목:**

| 항목 | 타입 | 설명 |
|---|---|---|
| 이동 갱신 | 스위치 | ON 시 이동 감지하여 자동 갱신 |
| 갱신 이동거리 | 슬라이더 | 5~20km (5km 단위), 이동 시 갱신 기준 거리 |

> 갱신시간(1,2,9,12,16,19시 KST)에는 항상 갱신됩니다. 이동 갱신을 OFF로 설정하면 갱신시간에만 호출합니다.

## Opinet API 키 발급 방법

1. [Opinet](https://www.opinet.co.kr) 접속 → 회원가입/로그인
2. **유가관련정보** → **유가정보API** → **무료 API 이용신청**
3. 신청 완료 후 발급된 `API 코드`를 API 키 항목에 입력

## Tmap API 키 발급 방법

1. [SK open API](https://openapi.sk.com) 접속 → 회원가입/로그인
2. **대시보드** → **앱** → **앱 만들기**
3. 생성된 `App Key` 값을 Tmap API 키 항목에 입력

> 무료 플랜 기준 월 30,000건 제공 (주유소 20개 × 하루 6회 갱신 ≈ 월 3,600건)

## VWorld API 키 발급 방법 (선택)

VWorld 지오코딩 API로 주유소의 정확한 WGS84 좌표를 획득합니다.
API 키를 입력하지 않아도 기본 제공 키로 **일 최대 50회**까지 이용 가능합니다.
개인 키 발급 시 무제한 사용 가능하며, 조회된 좌표는 자동으로 공유 캐시에 저장됩니다.

1. [VWorld 데이터 API](https://www.vworld.kr/v4po_main.do) 접속 → 회원가입/로그인
2. 상단 **오픈API** 메뉴 → **인증키 발급**
3. API 이름: `opinet-geocode`, 서비스: `주소 → 좌표 변환` 선택
4. 발급된 `API KEY`를 복사

> VWorld 무료 플랜 기준 **일 40,000건** 제공.
> 미입력 시 기본 공유 키로 일 50회 제한 (캐시 히트 시 제한 소모 안 함).
> 개인 키 입력 시 제한 없음 + 조회 결과는 자동 공유 캐시에 저장되어 다른 사용자도 혜택.

**통합구성요소에 VWorld API 키 설정:**

`설정 → 기기 및 서비스 → 오피넷 주유소 기기 → 설정 → API 키` 에서 VWorld API 키를 입력하세요.

## 센서 정보

`오피넷 주유소` 기기 아래 1~10위 센서가 생성됩니다.

**상태(state) 형식:**
- 거리 ON: `SK에너지: 1,984원 (4.1km)`
- 거리 OFF: `SK에너지: 1,984원`
- 주행거리순 사용 시: Tmap 실주행거리 우선 표시

**속성(Attributes):**
- `주유소명`, `가격`, `주소`, `간략주소`, `브랜드`, `거리`, `순위`

### API 사용량 센서

`sensor.opinet_price_api_usage` — 현재까지의 API 호출 횟수입니다.
다중 인스턴스 환경에서는 인스턴스별로 별도 생성됩니다.

**상태(state):** `오피넷 26회 | VWorld 5회 | Tmap 3,600회`

**속성:**
- `opinet_calls`, `vworld_calls`, `tmap_calls`

> 카운터는 매일 0시(오피넷/VWorld), 매월 1일 0시(Tmap) KST 기준 자동 리셋됩니다.

## 지도 카드

주유소 위치를 지도에 표시하려면 [Opinet Price Dashboard](https://github.com/Loa0/ha-opinet-price-dashboard)의 `custom:opinet-map-card`를 사용하세요.

> v1.8.0+ 부터 VWorld 지오코딩 API로 ±5~10m 정확도의 좌표가 제공되며, 조회된 좌표는 공유 Redis 캐시에 저장되어 이후 API 호출 없이 즉시 로드됩니다.

## 업데이트 방법

HACS에서 최신 버전 Redownload 후 **통합구성요소 삭제 → 재등록**을 권장합니다.

## 면책사항

본 통합구성요소는 Opinet, Tmap, VWorld의 공식 제품이 아니며, 각 사와 어떠한 제휴 관계도 없음을 알려드립니다.

- API 사용량 및 과금은 사용자님께서 직접 발급받으신 API 키를 기준으로 집계됩니다.
- 표시되는 유가 정보는 Opinet API의 갱신 주기에 따르므로, 실제 주유소 가격과 시차가 있을 수 있습니다.
- VWorld, Tmap 등 제3자 API의 점검이나 장애로 인해 일부 기능이 일시적으로 제한될 수 있습니다.
- 본 통합구성요소 사용으로 발생하는 모든 책임은 사용자에게 있습니다.
